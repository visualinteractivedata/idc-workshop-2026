# IDC 2026 Workshop: Data Vis Activities with Children

This is the official website for IDC 2026 workshop -- Data Vis Activities for Children: Hands-on Exploration of Activities & Challenges around Visual, Physical and Tangible Representations for Play and Data Literacy  

## How to work on this site locally

- Make sure you have [Ruby and Jekyll installed](https://jekyllrb.com/docs/installation/)
- Clone this repo to your computer
- Open a command line in that folder, run `bundle update` then `bundle exec jekyll serve`
- The website should now be available at `localhost:4000`
